# Loop Cam

Loop Cam is Loop Studio's branded fork of VDO.Ninja for bringing remote cameras, microphones, and screen shares into OBS and other studio software.

## Users

- **Directors** — create rooms, manage guests, control scenes
- **Performers / guests** — share camera, mic, or screen into a session
- **Solo streamers** — quick camera or screen publish without a full room

## Core flows

1. **Home** — pick Create a Room, Share Camera, or Share Screen
2. **Share Camera** — preview, pick devices, start stream
3. **Share Screen** — pick display, optional mic mix, start
4. **Create a Room** — name/password/options, enter as director or participant
5. **Live session** — full-screen video, floating control bar, copy browser-source URL

## Design intent

Dark, low-distraction tool UI. Familiar broadcast affordances. Lucide icons on upstream Line Awesome hooks. Polish should feel like a focused pro tool, not a marketing site.
